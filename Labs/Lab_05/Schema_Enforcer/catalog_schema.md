| Column Name | Required Data Type | Brief Description                                     |
| :---        | :---               | :---                                                  |
| `name`      | `VARCHAR(100)`     | Instructor’s full name.                               |
| `role`      | `VARCHAR(50)`      | Instructor’s role in the course (e.g., "Primary", "TA"). |
| `course_id` | `VARCHAR(10)`      | Course identifier (e.g., "DS2002").                   |
| `title`     | `VARCHAR(150)`     | Full course title.                                    |
| `level`     | `INT`              | Course level/category (e.g., 100, 200, 300, 400).     |
| `section`   | `VARCHAR(10)`      | Section identifier (e.g., "001").                     |
